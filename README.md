# «Место» / Бэкенд и фронтенд

Это бэкенд (а скоро будет и фронтенд) проекта Mesto.

Mesto — это страница с красивыми карточками. На ней можно зарегистрироваться, обновлять пользовательские данные, создавать и удалять карточки и даже ставить лукасы чужим.

Проект существует и в других ипостасях: [на чистом JS](https://github.com/ooohrayyy/mesto), [на React](https://github.com/ooohrayyy/mesto-react), [на React с регистрацией и авторизацией](https://github.com/ooohrayyy/react-mesto-auth). Нынешняя будет самой полной в плане функциональности. 

## Технологии

Проект работает на Node.js и express.js с библиотекой mongoose для связки с базой данных MongoDB.

## Планы на будущее

Главный план — поднять сервер в облаке и портировать в этот репозиторий весь фронтенд, чтобы проект получился полностью завершённым.
