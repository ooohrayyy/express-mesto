# «Место» / Бэкенд и фронтенд

Это бэкенд учебного проекта Mesto.

Mesto — это страница с красивыми карточками. На ней можно зарегистрироваться, обновлять пользовательские данные, создавать и удалять карточки и даже ставить лукасы чужим.

Проект существует в нескольких ипостасях: [на чистом JS](https://github.com/ooohrayyy/mesto), [на React](https://github.com/ooohrayyy/mesto-react), [на React с регистрацией и авторизацией](https://github.com/ooohrayyy/react-mesto-auth).

А вот [полная версия на «Реакте»,](https://github.com/ooohrayyy/react-mesto-api-full) в которой сомещены фронтенд и бэкенд.

## Технологии

Проект работает на Node.js и Express с библиотекой mongoose для связки с базой данных MongoDB.
